Залитить на гит:
	git status - чекнуть файлы которые были измененены. 
	git add .  - добавить все измененные файлы в список изменений
	git commit -m "текст коммита"  
	git push - залить коммит на гит


Скачать изменения:
	git pull - обновить все изменения.
